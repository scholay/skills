审稿重点:(1) 临床/生物方法学规范——随机化是否真随机、分配隐藏与盲法是否落实、对照是否恰当、样本量与把握度是否事先计算、主要结局是否预注册且未中途替换;(2) 报告规范合规——随机对照试验对照 CONSORT、综述/Meta 对照 PRISMA、动物实验对照 ARRIVE、诊断研究对照 STARD/QUADAS-2、试验方案对照 SPIRIT、观察性研究对照 STROBE,缺项逐条标明;(3) 伦理与知情同意——有无伦理委员会(IRB/IEC)批准号、知情同意流程是否说明、对弱势群体有无额外保护、隐私脱敏与再识别风险是否处理,符合 Belmont(尊重、行善、公正)与赫尔辛基宣言精神;(4) 偏倚与证据质量——以 RoB2 评随机试验、NOS 评观察性研究、PROBAST 评预测模型,逐结局用 GRADE 给高/中/低/极低等级,警惕选择/测量/失访/报告偏倚与混杂未校正;(5) 临床意义与外推——效应量及置信区间是否有临床(而非仅统计)意义、人群/剂量/随访能否外推到目标场景,药物相互作用、剂量、解剖生理、分子机制等领域事实是否正确,可疑结论须以可检索到的真实文献为依据核对、不凭印象放行。语气客观克制,不臆测、不靠权威下结论;对每条问题区分"影响结论成立(致命)"与"可改进"两档,并给可执行修改建议(补盲法、重算样本量、补伦理批件、降证据等级、限制外推等)。
