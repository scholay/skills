# 学术写作质量(Prism)

改的是给同行读的论文正文。**保持 LaTeX 结构与所有 `\cite{}` 位置不变**,只动语言。

## 句段层
- **主题句先行**:每段第一句直接给论点(claim-first),后续句子支撑它;段落只讲一个论点。
- **主动语态**为主;句子均长 ~20 词,过长(>40 词)拆开。
- **论证链可定位**:claim → evidence(指向具体图/表/数字),不空泛断言。
- 术语 / 记号**全文一致**;首次出现的缩写先展开。

## 章节修辞动作(写/补对应章节时套用)
- **引言**:研究动机(stakes)→ 问题缺口 → 关键想法 → 贡献清单 → 结果预览。
- **方法/设计**:动机直觉先行,再给形式化;别堆公式不解释。
- **实验**:设置锚定 → 与基线 head-to-head 对比 → 分解深入 → 数字的"所以呢"(takeaway)→ 消融 → 稳健性。
- **相关工作**:按主题分组,点明与本文的差异/定位,不流水账罗列。
- **摘要**:背景一句 + 做了什么 + 关键结果(带数字)+ 意义;别写成目录。

## 压缩(用户嫌啰嗦 / 要精简时)
- 删冗余形容词、合并近义段落、缩句、删对同行读者属常识的"教程式"铺垫、把陈述句转成 claim-first。
- 目标 30–50% 的删减仍保留全部论点 = 健康压缩。

## 去 AI 腔(de-AI,让文字像人写的)
- 删空洞套话:"众所周知 / 值得注意的是 / 在本节中,我们将 / 综上所述 / It is worth noting / In conclusion"。
- 删过度自信与机械排比("首先…其次…再次"那种空壳并列);删滥用的 em-dash(—)。
- 删低信息量的"承上启下"句;每句都要承载新信息。
- 框架/定位**跨章节稳定**,不要中途换提法(framing drift 是被拒稿的相关因素)。

> 注意:本 skill 只改语言质量,**不改学科规范/排版**——那分别走 [prism-discipline-conventions] 与 [prism-latex-format]。
